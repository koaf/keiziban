掲示板 画像アップロード対応/スマホ対応/PHP7対応/Akismet・reCAPTCHA対応

特徴

  • PHPが動作するレンタルサーバー等に設置可能。
  • HTMLファイルを修正するだけでデザイン変更できます。
  • スマホ/タブレット対応レスポンシブデザイン。(bootstrap)
  • ファイルアップロード、画像縮小機能。
  • Akismet、reCAPTCHAによる対スパム投稿機能。
  • 禁止ワード機能。
  • 最新の投稿をRSS配信。
  • スペイン語、ロシア語、中国語など、日本語以外の投稿も受け付けます。(UTF-8版)
  • PHP7対応。Linuxサーバー、Windowsサーバーに対応。

注意

  1. 削除依頼・苦情に対応してください

    掲示板運営者のメールアドレスなど連絡先を表示し、投稿に対する削除依頼や苦情に対応してください。
    掲示板運営者を隠している場合、貴殿に初めて連絡してくる人がレンタルサーバー事業者・ 警察官・法務局などになって、有害投稿を放置した責任を問われる可能性があります。
    例)

    この掲示板の投稿により不利益を受けている方、 投稿の削除を依頼される方はご連絡ください。 メールアドレス: taro@example.com (←貴殿のメールアドレス)

  2. 継続的に維持管理してください
    • 検索エンジンに掲載しないようにしてください。偽ブランド品宣伝などの迷惑投稿者が来るようになり、管理が困難になります。
    • 投稿がなく時間が経過すると、安全のため管理者以外は投稿はできなくなります。管理者が投稿すると、また誰でも投稿できるようになります。
    • バージョンリリース日から3-4年経過すると、掲示板は使用できなくなります。この場合は、最新バージョンに更新してください。
    • 管理できなくなったら、忘れないうちに掲示板を削除してください。

設置方法

  1. ダウンロードしたzipファイルを展開する
  2. レンタルサーバースペースにファイル一式をアップロードする
  3. ウェブブラウザでbbs.phpにアクセスする。
  4. 3でファイル書き込みに関するメッセージが出た場合は、 dataフォルダ・filesフォルダに書き込み許可属性を設定する。
